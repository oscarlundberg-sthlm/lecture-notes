## Lecture Notes

Sometimes teachers hand out their presentation beforehand.
I wanted a way to have the PDF to the left, and notes for each page/slide to the right - to be able to take notes directly linked to the relating slide.

## 100% local

Just load a PDF locally and save it as a project (\*.lectnote). All's local - no accounts or logins.

## Support

I only made this work on my computer (M1 Mac). If you have the know how, feel free to make it work for yours.

## Where's the app?

1. Open a terminal in the root folder (the top lecture-notes folder).
2. Build the app by writing `yarn build`.
3. When it's done, a folder will open. The app is within the Lecture Notes-darwin-arm64 subfolder.
